ACTIVITY 3

Control Inetrface


Some high level features of the control interface is smart parking system, effecient security system, fire prevention system (incase of an emergency), logging and backing the data or providing realtime data to users for efficent functioning. The users of these interfaces will be the resisdents, visitors, staff members, security and maintainence staff. Some of the sensors that are required for the systems are -

  • Temperature and humidity Sensor
  • Weight Sensor
  • Motion Sensor
  • Rfid System
  • Smoke Sensor
